# Changelog

## 1.2.0

- 修复 AstrBot 将多词游戏名截断为首个单词的问题。
- 地区参数改为前置 `-地区`，游戏名及特殊字符保持原样传递。
- 增加可选 LLM 游戏名和未知中文地区校正，支持可配置重试。
- 取消不相关 Steam 搜索首项兜底，避免返回错误游戏。

## 1.1.0（已撤回）

- 增加当前价、促销历史、全球区价、基础资料和详细资料模式。
- 使用 Steam 官方当前价，并从小黑盒历史价格点推导促销事件。
- 改用异步 HTTP 客户端，移除失效的小黑盒 mobile detail 接口。
- 增加 AstrBot 市场发布前检查、Logo 和权利声明。
- GitHub Release 和标签因命令参数绑定问题撤回，未提交 AstrBot 市场。

## 1.0.0

- 首次发布独立的小黑盒 Steam 价格查询插件。
